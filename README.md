Si estás aquí es por que parece ser que te ha interesado un poco mi minijuego y me alegro mucho de que lo intentes probar.
Para poder utilizarlo necesitas tener descargado Visual Studio Code junto con la extensión de Live server.
Los pasos a seguir una vez descargada e instalada la extensión es:
Archivo -> Abrir carpeta -> abrimos haciendo doble click en el archivo Juego.html y justo abajo en la esquina inferior derecha del programa tiene que aparecer en letras pequeñas "Go Live" una vez que seleccionemos eso se abrirá el juego en tu navegador.
Ahora solo queda pulsar el botón de START ¡Y a recolectar peces!
